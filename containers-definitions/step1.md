A continuación se muestran beneficios al utilizar contenedores
* Aplicaciones con tolerancia a fallos
* Rápido despliegue
* Poco consumo de recursos
* Almacenamiento efímero o persistente
* Altos niveles de seguridad

Los contenedores aunque cuentan con muchos beneficios, se debe de considerar que no son capaces de realizar todas las actividades que se pueden realizar algún sistema operativo, a continuación se muestran beneficios de sistemas operativos y contenedores
Beneficios de sistemas operativos
* Todos los recursos del sistema operativo están disponibles para las aplicaciones.
* Herramientas de gestión y seguridad establecidas.
* Controles de seguridad más sencillos y controlables.
* Administración de Hardware físico o virtual

Beneficios de los Contenedores
* Recursos de gestión de TI reducidos.
* Gestión de datos comprimidos.
* Más rápido en la gestión de aplicaciones.
* Actualizaciones de seguridad reducidas y simplificadas.
* Menos código para transferir y migrar cargas de trabajo.
